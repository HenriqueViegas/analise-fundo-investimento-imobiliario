# investimento-fundos-imobiliarios
Este projeto automatiza a coleta e análise de dados de Fundos Imobiliários listados na B3. O objetivo é filtrar ativos com base no Dividend Yield e Preço sobre Valor Patrimonial para tomada de decisão em investimentos.

Funcionalidades
Web Scraping: Coleta automatizada de dados de portais financeiros (ex: Status Invest, Funds Explorer).

Data Cleaning: Tratamento de dados nulos, conversão de tipos e limpeza de strings.

Análise Quantitativa: Cálculo de indicadores como:

P/VP (Preço sobre Valor Patrimonial).

Dividend Yield (DY) acumulado.

Vacância física e financeira.

Liquidez diária.

Ranking: Geração de uma lista dos FIIs que atendem aos critérios pré-definidos de "bons parâmetros".
